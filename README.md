# 婚活リアリティ電卓 (konkatsu-reality-calc)

希望する結婚相手の条件を入力すると、**その条件に当てはまる未婚者が現実に何人いるか**を公開統計からざっくり推定するWebアプリのプロトタイプです。

## コンセプト

婚活の苦しさの多くは「相手がいない」ことではなく、**自分の希望と現実の市場のズレに気づけないまま消耗すること**にあります。
このツールは、条件 → 母数 → 希少度 → 出会い期待値を可視化し、「どの条件を緩めると母数が一番増えるか」まで提示することで、感情の問題を“計算できる問題”に変換します。

## できること

- 条件（性別・年齢・エリア・年収・身長・学歴）を入力 → **該当する未婚者の推定人数**
- **絞り込みファネル**：どの条件が一番母数を狭めているかを棒グラフで可視化
- **希少度**：「同世代未婚の◯人に1人」
- **出会い期待値**：今の生活ペースだと年に何人出会えるか
- **改善レバー**：条件を一段緩めたときの母数増加率（効果の大きい順 top3）
- 結果のシェア（LINE等）

## 推定ロジック

```
母数 N = 地域の対象性別人口(年齢範囲)
        × 未婚率(年齢)              ← 国勢調査
        × P(年収 ≥ 条件 | 年齢・性別) ← 賃金構造基本統計調査
        × P(身長 ≥ 条件 | 性別)      ← 国民健康・栄養調査(正規分布近似)
        × P(学歴 = 条件)            ← 就業構造基本調査
```

各条件は独立とみなして掛け合わせる簡易モデルのため、相関のある条件（年収×学歴等）では実際よりやや少なめに出ます。傾向把握用の概算です。

## データ出典

- 総務省「国勢調査(2020)」— 配偶関係・年齢・人口
- 厚生労働省「賃金構造基本統計調査」— 年収分布
- 厚生労働省「国民健康・栄養調査」— 身長分布
- 総務省「就業構造基本調査」— 学歴

## 使い方

`index.html` をブラウザで開くだけ。ビルド不要・依存ライブラリなしの単一ファイルです。
